Quando um elemento é flutuado, seu pai não o contém mais porque o flutuante é removido do fluxo. O elemento flutuante está fora do fluxo natural, portanto, todos os elementos de bloco serão renderizados como se o elemento flutuante não estivesse lá, portanto, um contêiner pai não será totalmente expandido para conter o elemento filho flutuante.
